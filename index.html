Sistema de Venta de Libros

Plataforma web y API para gestión y comercialización de libros

Sistema de venta de libros desarrollado con Node.js y Express, utilizando Handlebars (HBS) para la interfaz web y PostgreSQL como base de datos principal. La arquitectura está organizada en controladores, rutas y middleware personalizados, permitiendo un manejo claro y escalable de la lógica de negocio.

Características Principales
  • Interfaz web con Handlebars (HBS) para navegación, catálogo y detalle de libros.
  • CRUD completo de libros (crear, listar, editar y eliminar).
  • Conexión a base de datos PostgreSQL mediante consultas parametrizadas.
  • API REST independiente para consumo de servicios desde Postman u otros clientes.
  • Separación clara entre vista (HBS) y API, evitando conflictos entre rutas.
  • Uso de middleware para validaciones, autenticación y control de acceso.
  • Estructura modular con controllers, services y rutas por responsabilidad.
Tecnologías Utilizadas
Node.js Express PostgreSQL HBS / Views JavaScript Middleware REST API
